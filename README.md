# Vehicle-Number-Plate-Detection
This project aims to extract the registration number of a vehicle from the image captured by the CCTV camera.

---

## Dependencies
- Python
- OpenCV
- Numpy
- RegEx

---

## How to Use

- Clone this repo
```html
git clone https://github.com/Pritam-Sethuraman/Vehicle-Number-Plate-Detection.git
```

- Run main.py
```html
python main.py
```
